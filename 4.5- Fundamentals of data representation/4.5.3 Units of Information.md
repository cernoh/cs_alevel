## 4.5.3.2 Units
Quantites can be represented by prefixes. 

prefixes go up in :
Binary -> powers of two
decimal -> powers of ten

**Binary**

| Prefix    | Value    |
| --------- | -------- |
| Kibi (Ki) | $2^{10}$ |
| Mebi (Mi) | $2^{20}$ |
| Gibi (Gi) | $2^{30}$ |
| Tebi (Ti) | $2^{40}$ |


**Decimal**

| Prefix   | Value   |
| -------- | ------- |
| Kilo (K) | $10^3$  |
| Mega (M) | $10^6$  |
| Giga (G) | $10^9$  |
| Tera (T) | $10^12$ |
